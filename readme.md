# Easy Learn Digital Signal Processing

## 项目简介
本项目旨在为非通信相关专业的学习者提供信号与系统的知识以及相关的数学基础，如傅里叶变换与z变换。通过python代码进行数字信号处理，使学习者能够更快上手数字信号处理。
